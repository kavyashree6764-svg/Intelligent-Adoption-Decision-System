# Intelligent Adoption Decision System using ML & XAI

##  Description
AI-based system for adoption decision-making using Machine Learning and Explainable AI.

## Features
- Compatibility Analysis
- Fraud Detection
- Adaptive Decision System
- Explainable AI (XAI)
- Dashboard Visualization

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Chart.js

##  How to Run

pip install flask flask-cors  
python app.py  

Open index.html in browser
